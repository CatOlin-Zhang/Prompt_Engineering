#  Study Companion (RAG System)

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![Ollama](https://img.shields.io/badge/Ollama-Supported-green.svg)](https://ollama.ai/)

This is an intelligent learning companion system based on Retrieval-Augmented Generation (RAG) technology. The project aims to provide students with precise academic consulting, course recommendations, and campus life guidance using a locally deployed Large Language Model (LLM). The system supports multiple document formats (PDF, DOCX, Excel, TXT) and integrates a comparison mechanism for three retrieval strategies along with an automatic evaluation function.

---

##  Key Features

- **Multi-Format Data Support**: Supports loading `.txt`, `.pdf`, `.docx`, and `.xlsx` files. It can intelligently process Excel table data by converting it into JSON objects.
- **Three Retrieval Strategies Comparison**:
  - **Lexical Retrieval**: Keyword matching based on TF-IDF.
  - **Neural Retrieval**: Semantic vector matching based on Sentence Transformers.
  - **Hybrid Retrieval**: Weighted combination of the above two methods, typically yielding the best results.
- **LLM Judge System**: An integrated evaluation module that automatically compares the quality of answers generated by the three strategies and selects the best one.
- **Local Model Deployment**: Runs local models (such as Gemma 3) based on Ollama, ensuring data privacy and fast response.
- **Conversation Memory Management**: Supports multi-turn dialogue. The system maintains historical context to understand user intent.

##  Environment Configuration & Installation

### 1. Install Dependencies

Please run the following command in your terminal or Jupyter Notebook to install the required Python libraries:

```bash
  pip install torch numpy pandas PyPDF2 docx sentence-transformers scikit-learn ollama tiktoken

```
A requirements.txt file is also provided in the directory
```bash
  conda install --file requirements.txt
```
### 2. Install Ollama Models
- This project requires two models: one for local generation and one for cloud evaluation (Judge). 
Ensure the Ollama service (ollama serve) is running and pull the following models:
- If you do not have permission to call the cloud model, please disable the LLM Judge function.
```bash
  # Local Generation Model
  ollama pull gemma3:4b
  
  # Judge Model (Used for evaluating answer quality)
  ollama pull gemma3:4b-cloud
  ```
### 3. Prepare Data
Create a folder named data in the project root directory and place your knowledge base files (PDF, Word, Excel, etc.) into this folder.
```text
project_root/
├── the_notebook_version.ipynb
├── data/
│   ├── course_info.pdf
│   └── regulations.docx
```

### 4. System Architecture & Modules

####  Data Processing Module (Data Loading & Chunking)
- Function: Automatically traverses the data directory and calls corresponding read functions based on file extensions.
- Smart Chunking: For Excel/JSON data, the system attempts to maintain object integrity, avoiding the splitting of single records.
####  Retriever Module (RAG Retriever)
- TF-IDF: Used for exact keyword matching.
- Embedding: Uses the all-MiniLM-L6-v2 model to vectorize text and calculate cosine similarity.
- Hybrid: Normalizes (Min-Max Scaling) and weights the scores from both retrieval methods.
####  Generator Module (Ollama Generator)
- Prompt Engineering: Built-in prompt templates include role settings, Few-Shot examples, and Chain-of-Thought (CoT) to guide the model's step-by-step reasoning.
- Token Counting: Uses tiktoken to accurately calculate the Token count of input and output.
####  Judge Module (LLM Judge)
Evaluation Criteria: 
- Fidelity: Whether the answer adheres to the facts in the reference material.
- Completeness: Whether all key points are covered.
- Restraint: Whether the response contains hallucinations or redundant information.
### 5. Notes
- Ollama Service: Please ensure the Ollama application is running in the background before executing the code.
- VRAM Requirements: Running gemma3:4b requires a certain amount of GPU VRAM or system memory. If performance is slow, please check your hardware resources.
- Excel Processing: Excel files are converted into JSON strings for processing. Ensure the table structure is clear to achieve the best retrieval effect.
